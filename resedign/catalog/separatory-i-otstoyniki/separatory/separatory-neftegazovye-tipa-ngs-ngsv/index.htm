Сепараторы нефтегазовые типа НГС, НГСВ

Нефтегазовые сепараторы широко используются в нефте- и газоперерабатывающей промышленности. Их основным назначением является разделение нефтяной смеси на нефть, газ и воду, а также очистка попутного газа от примесей. Данный тип оборудования может применяться на всех ступенях подготовки нефти и газа.

Существует два типа нефтегазосепараторов:

·         Сепаратор нефтегазовый типа НГС

·         Сепаратор нефти и газа типа НГСВ (со сбросом воды)

Технические характеристики

Сепаратор нефти и газа НГС

Нефтегазовый сепаратор НГС представляет собой горизонтальный стальной сосуд цилиндрической формы. Внутренняя конструкция емкости представлена отбойником для грубого разделения поступающей смеси нефти и газа, а также вертикальной перегородкой из вытяжных листов, которая обеспечивает выравнивание скорости потока. Сепаратор НГС также оснащен струнным каплеулавливателем, пеногасящей насадкой и штуцерами, необходимыми для входа и выхода поступающей смеси.

Выделяют два вида данной модели сепаратора:

-        для работы с депульсаторами (узлами предварительного отбора газа)

-        для работы без депульсаторов

Сепараторы нефтегазовые НГСВ

Данный вид нефтегазосепараторов предназначен для разделения водонефтяной смеси на нефть газ и воду.

Нефтегазовый сепаратор со сбросом воды НГСВ представляет собой металлический горизонтальный сосуд цилиндрической формы, установленный на опоры. Изнутри аппарат также может быть покрыт специальным антикоррозийным составом.

Сепараторы НГСВ могут быть изготовлены как с пеногасителем так и без него. По требованию клиента аппарат также может быть термообработанным и иметь крепления для теплоизоляции.

Нефтегазосепараторы типа НГС и НГСВ выпускаются в следующих вариантах материального исполнения:

-        Сталь Ст3 (рекомендована для температурных показателей не ниже - 30°C)

-        Сталь 09Г2С-12 (рекомендована для температурных показателей не ниже - 40°C)

-        Сталь 09Г2С-15 (рекомендована для температурных показателей не ниже - 60°C)

-        Нержавеющая сталь 12Х18Н10Т (аналог AISI)

Также возможны два варианта климатического исполнения:

•        поставляется в регионы с умеренным и холодным климатом - УХЛ по ГОСТ 15150;

•        поставляется в регионы с холодным климатом - ХЛ по ГОСТ 15150.
